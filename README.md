# Aryu Blog, REST API
Simple and practice project aimed at skill enhancement, utilizing and combining the various technologies listed below:

- **Python, Django Framework**
- **Django REST Framework**
- **Swagger, Redoc**
- **PostgreSQL, Redis**
- **Celery, RabbitMQ**
- **Simple JWT**
- **Gunicorn, Nginx**
- **Git, Gitflow, Docker**
